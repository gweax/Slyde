Slyde

Slyde is a web based presentation tool that uses advanced technologies like HTML5 and CSS3.

Transitions

There are several effects to switch from one slide to another. Try changing the class name of the body element.

Transitions - CSS only

CSS3 is a cool way to allow for transition effects. There is no Javascript involved in that part. However, for more fancy effects, some Javascript might be required.

Navigation

The navigation elements on the slides are generated automatically. Therefore, you do not have to adapt every slide if you want to insert a new slide between existing ones.

Key navigation

You can navigate through the slides either by using the generated link elements or by using the keyboard.

Try the Page Down, Page Up, Home and End keys as well as the left and right arrow keys.

Linking to a specific slide

Navigation is handled through the location's hash value, so you can directly link to any slide within a presentation. Note that this - again - is done by CSS only without any Javascript.

Point by point

Sometimes you wish several elements like list items to appear one after the other. This is easy, just add class="increment" to those elements.

Browsersupport

Works best in Chrome and Safari. Animations are slow in Opera, but technically, it works. In Firefox 3.6 transitions do not work for some reason. Besides, Firefox lacks some CSS3 support.

It does not work in IE up to version 8. I do not care.

Yet to come

Currently, the font size is fixed. This is bad, when the browser window is small (or large). Adapting font size to the available space is on the to-do list.
